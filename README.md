# SQL Trainer Platform

Учебная платформа для практического освоения SQL с возможностью загрузки тематических баз данных преподавателем, мониторинга активности студентов и экспорта результатов с электронной подписью.

## 📋 Функциональность

### Для студентов:
- **Выбор базы данных** - переключение между несколькими учебными базами
- **Выполнение SELECT запросов** с таймаутом 3 секунды
- **Просмотр структуры базы** - список всех таблиц в выбранной базе
- **Анализ плана выполнения** - визуализация EXPLAIN ANALYZE для каждого запроса
- **Экспорт результатов** - скачивание результатов запроса в CSV формате
- **Защита от подделки** - CSV файл содержит метаданные и электронную подпись:
  - Время выполнения запроса
  - Количество строк
  - Дата и время генерации
  - Уникальная подпись (хеш) для верификации
- **Ограничения безопасности**:
  - Таймаут выполнения: 3 секунды
  - Максимальное количество строк: 1000
  - Только SELECT запросы

### Для преподавателя:
- **Защищенный вход** - единый пароль через переменную окружения
- **Загрузка баз данных** - загрузка SQL-скриптов через веб-интерфейс
  - Автоматическое извлечение имени базы из скрипта
  - Проверка расширения (.sql)
  - Ограничение размера (10 MB)
  - Защита от опасных SQL команд
- **Управление базами** - просмотр и удаление существующих баз
- **Мониторинг сессий** - отслеживание активных сессий студентов в реальном времени
- **Просмотр запросов** - история последних запросов студентов
- **Визуальный прогресс** - индикатор загрузки при выполнении скриптов
- **Детальный лог** - пошаговое отображение процесса создания базы

## 🛠 Технологический стек

- **Backend**: Java 17, Servlets, JSP, JDBC
- **Frontend**: HTML5, CSS3, JavaScript (vanilla)
- **База данных**: PostgreSQL 15+
- **Сборка**: Maven
- **Сервер**: Apache Tomcat 7
- **Контейнеризация**: Docker, Docker Compose
- **Безопасность**:
  - Переменные окружения для чувствительных данных
  - Фильтры для защиты страниц преподавателя
  - Сессии для авторизации
  - Ограничения PostgreSQL на уровне ролей
