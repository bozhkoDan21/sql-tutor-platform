# SQL Tutor

Учебная платформа для практического освоения SQL.

## Структура проекта

- `/src/main/java` - Java классы
- `/src/main/resources` - конфигурационные файлы
- `/src/main/webapp` - фронтенд (JSP, CSS, JS)

## Запуск

1. Создайте базу данных в PostgreSQL
2. Настройте подключение в `application.properties`
3. Запустите через Maven: `mvn tomcat7:run`
4. Откройте браузер: `http://localhost:8080`

## Функциональность

- Выполнение SELECT запросов к учебным базам
- Анализ плана выполнения (EXPLAIN)
- Загрузка новых баз данных преподавателем
