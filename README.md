# SQL Tutor Platform

Учебная платформа для практического освоения SQL с возможностью загрузки тематических баз данных преподавателем и мониторинга активности студентов.

## 📋 Функциональность

### Для студентов:
- Выполнение SELECT запросов к учебной базе данных
- Просмотр структуры базы (список таблиц)
- Анализ плана выполнения запросов (EXPLAIN ANALYZE)
- Ограничения по времени (30 сек) и количеству строк (1000)

### Для преподавателя:
- Защищенный вход по паролю (переменная окружения)
- Загрузка новых SQL-баз данных через веб-интерфейс
- Управление существующими базами (удаление)
- Мониторинг активных сессий студентов в реальном времени
- Просмотр последних запросов студентов

## 🛠 Технологический стек

- **Backend**: Java 17, Servlets, JSP, JDBC
- **Frontend**: HTML5, CSS3, JavaScript (vanilla)
- **База данных**: PostgreSQL 15+
- **Сборка**: Maven
- **Сервер**: Apache Tomcat 7
- **Безопасность**: Переменные окружения, фильтры, сессии
